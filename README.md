# 🧰 Terraform AWS Labs

Small hands-on Terraform labs to practice **AWS resource provisioning**.

---

## 📂 Labs

- **Challenge 1 – IAM Users** 🛡️: Create IAM users with environment tags.  
- **Challenge 2 – Bulk S3 Buckets** 📦: Provision multiple buckets with `for_each`.  
- **Challenge 3 – Multi-Env S3** 🌐: Dynamic bucket names & tagging.  
- **Challenge 4 – S3 File Upload** 📤: Upload local files into S3.  
- **Challenge 5 – DynamoDB Tables** 💾: Create tables with GSIs.  
- **Challenge 6 – Insert Data** 🔄: Upload sparse items and extract ARNs.  

---

## ⚡ Skills

- Terraform: variables, loops, tags, outputs  
- AWS: IAM, S3, DynamoDB  
- IaC & automation best practices  

---
